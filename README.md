# MyPortfolio Application

> This is a Ruby on Rails 5 application that allows the users to create their own portfolios 

### Running with Subdomains 
. rails s -p 3002
### Features 

- Real time chat engine for comments 
- Blog 
- Portfolio 
- Drag and drop interface
